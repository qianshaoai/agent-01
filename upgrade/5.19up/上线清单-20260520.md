## 最易漏的 5 个（敲黑板）

1. **A1 vector 扩展启用** —— SQL 第一行 `CREATE EXTENSION IF NOT EXISTS vector` 在没权限时会直接报错，必须先在 Dashboard 启用。
2. **B2 embedding 供应商要在线上库再配一遍** —— 本机 dev 配的不会同步到生产。
3. **B1 Fix 3 rekey 不可逆** —— 跑前必须库备份点。
4. **C2 知识库智能体不要选 glm-4-flash** —— 模型能力压不住 KB，体验直接崩。
5. **代理 fake-ip 别带到 prod** —— 本地代理（Clash 等）的 fake-ip 模式只影响开发；生产用真实 DNS，不必担心。但**本地验收前**得修，否则连 `fetch failed` 都搞不清是代码还是网络。
