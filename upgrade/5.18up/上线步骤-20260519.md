## 7. 回滚预案

| 出问题 | 应对 |
| --- | --- |
| 构建/部署失败 | Jenkins 重跑上一次成功构建；或 master `revert` 合并 commit 再推 |
| 忘建 ENCRYPTION_KEY 凭据 | 构建在 environment 阶段即失败，补建凭据后重跑，无脏数据 |
| rekey dry-run 报问题行 | 不勾 APPLY 就不写库，零影响；按报告人工处理后再跑 |
| rekey `--apply` 后发现异常 | 用步骤 3 的备份恢复对应表的 `*_enc` 列 |
| 加密相关整体异常 | crypto.ts fallback 链保证老密文仍可解，最坏回滚代码即可 |
