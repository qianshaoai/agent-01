# 体验版 · 阶段三行动方案 · 前端页面与路由分流

**日期**：2026-04-28
**前置**：阶段一、阶段二、阶段四均已完成
**目标**：体验账号登录后自动进入 `/trial`，能选智能体能流式聊天；体验账号访问其他页面被 server-side 跳回 `/trial`
**预计工时**：4 小时

---

## 1. 阶段目标

完成本阶段后系统状态：

- 登录接口响应体携带 `userType`
- 登录页根据 `userType === "trial"` 跳 `/trial`，否则保留原有跳转
- middleware 的 `USER_PROTECTED` 列表包含 `/trial`，未登录访问 `/trial` 跳 `/login`
- middleware 在用户登录校验通过后再做一次 trial 分流：trial 用户访问 `/`、`/agents` 等正式页面 → server-side redirect 到 `/trial`
- `/trial` 页面 UI 可用：智能体展示区 + 聊天输入框 + 流式回复气泡
- 不闪烁：trial 用户**不会**先看到正式页面再跳转

### 3.3 `middleware.ts` 路由保护

定位：`middleware.ts:5`

```ts

const USER_PROTECTED = ["/", "/agents", "/settings", "/user-agents"];

// 改为

```

定位：`middleware.ts:55-65` 用户路由保护块，在 `if (!payload || payload.type !== "user")` 下面追加 trial 分流：

```ts
if (
  USER_PROTECTED.some(
    (p) => pathname === p || pathname.startsWith(p + "/")
  )
) {
  const payload = await getPayloadFromRequest(req);
  if (!payload || payload.type !== "user") {
    return NextResponse.redirect(new URL("/login", req.url));
  }
  // ── 新增：trial 用户只允许停留在 /trial ──
  if (payload.userType === "trial" && !pathname.startsWith("/trial")) {
    return NextResponse.redirect(new URL("/trial", req.url));
  }
  return NextResponse.next();
}
```

### 3.4 `app/trial/page.tsx` UI

骨架（非强约束）：

```tsx
"use client";
import { useState, useEffect, useRef } from "react";
import { useRouter } from "next/navigation";
import { Bot, Send, Loader2 } from "lucide-react";

type TrialAgent = { id: string; name: string; description: string; avatar: string };
type Msg = { role: "user" | "assistant"; content: string };

export default function TrialPage() {
  const router = useRouter();
  const [agents, setAgents] = useState<TrialAgent[]>([]);
  const [activeId, setActiveId] = useState<string>("");
  const [messages, setMessages] = useState<Msg[]>([]);
  const [input, setInput] = useState("");
  const [streaming, setStreaming] = useState(false);
  const [loadErr, setLoadErr] = useState("");

  useEffect(() => {
    fetch("/api/trial/agents").then(async (r) => {
      if (r.status === 401) { router.push("/login"); return; }
      const d = await r.json();
      const list: TrialAgent[] = d.data ?? [];
      setAgents(list);
      if (list.length > 0) setActiveId(list[0].id);
      if (list.length === 0) setLoadErr("体验版未配置智能体，请联系管理员");
    });
  }, [router]);

  async function send() {
    const text = input.trim();
    if (!text || !activeId || streaming) return;
    setInput("");
    setMessages((m) => [...m, { role: "user", content: text }, { role: "assistant", content: "" }]);
    setStreaming(true);
    try {
      const res = await fetch("/api/trial/chat", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({ agent_id: activeId, message: text }),
      });
      if (!res.ok || !res.body) throw new Error("请求失败");
      const reader = res.body.getReader();
      const decoder = new TextDecoder();
      let buf = "";
      while (true) {
        const { value, done } = await reader.read();
        if (done) break;
        buf += decoder.decode(value, { stream: true });
        const lines = buf.split("\n\n");
        buf = lines.pop() ?? "";
        for (const line of lines) {
          if (!line.startsWith("data:")) continue;
          const json = line.slice(5).trim();
          if (!json) continue;
          try {
            const obj = JSON.parse(json);
            if (obj.delta) {
              setMessages((m) => {
                const copy = [...m];
                copy[copy.length - 1] = { ...copy[copy.length - 1], content: copy[copy.length - 1].content + obj.delta };
                return copy;
              });
            }
            if (obj.error) {
              setMessages((m) => {
                const copy = [...m];
                copy[copy.length - 1] = { role: "assistant", content: `（出错：${obj.error}）` };
                return copy;
              });
            }
          } catch {}
        }
      }
    } catch (e) {
      setMessages((m) => {
        const copy = [...m];
        copy[copy.length - 1] = { role: "assistant", content: "（网络错误，请重试）" };
        return copy;
      });
    } finally {
      setStreaming(false);
    }
  }

  return (
    <div className="min-h-screen bg-gradient-to-br from-[#f0f4ff] via-white to-[#f8f9ff] flex flex-col">
      <header className="px-6 py-4 border-b border-gray-100 bg-white/80 backdrop-blur">
        <h1 className="text-lg font-semibold text-gray-900">体验版</h1>
      </header>
      <main className="flex-1 max-w-3xl w-full mx-auto px-4 py-6 flex flex-col gap-4">
        {loadErr && <div className="p-3 bg-red-50 text-red-500 rounded-[10px] text-sm">{loadErr}</div>}

        {/* 智能体选择 */}
        {agents.length > 1 && (
          <div className="flex gap-2 flex-wrap">
            {agents.map((a) => (
              <button
                key={a.id}
                onClick={() => setActiveId(a.id)}
                className={`px-3 py-1.5 rounded-full text-sm border transition-colors ${
                  activeId === a.id ? "bg-[#002FA7] text-white border-[#002FA7]" : "bg-white text-gray-600 border-gray-200 hover:border-[#002FA7]"
                }`}
              >
                <Bot size={12} className="inline mr-1" /> {a.name}
              </button>
            ))}
          </div>
        )}

        {/* 单智能体场景：显示 description */}
        {agents.length === 1 && (
          <div className="p-3 bg-white rounded-[12px] border border-gray-100">
            <p className="text-sm font-medium text-gray-900">{agents[0].name}</p>
            <p className="text-xs text-gray-500 mt-1">{agents[0].description}</p>
          </div>
        )}

        {/* 消息流 */}
        <div className="flex-1 flex flex-col gap-3 overflow-y-auto">
          {messages.map((m, i) => (
            <div key={i} className={`flex ${m.role === "user" ? "justify-end" : "justify-start"}`}>
              <div className={`max-w-[80%] px-3 py-2 rounded-[12px] text-sm whitespace-pre-wrap ${
                m.role === "user" ? "bg-[#002FA7] text-white" : "bg-white border border-gray-100 text-gray-800"
              }`}>
                {m.content || (streaming && i === messages.length - 1 ? <Loader2 size={14} className="animate-spin" /> : "")}
              </div>
            </div>
          ))}
        </div>

        {/* 输入框 */}
        <div className="flex gap-2 items-end pt-2 border-t border-gray-100">
          <textarea
            value={input}
            onChange={(e) => setInput(e.target.value)}
            onKeyDown={(e) => { if (e.key === "Enter" && !e.shiftKey) { e.preventDefault(); send(); } }}
            placeholder="输入消息，Enter 发送，Shift+Enter 换行"
            rows={2}
            className="flex-1 resize-none border border-gray-200 rounded-[12px] px-3 py-2 text-sm focus:outline-none focus:border-[#002FA7]"
          />
          <button
            onClick={send}
            disabled={streaming || !input.trim() || !activeId}
            className="h-10 px-4 rounded-[12px] bg-[#002FA7] text-white text-sm font-medium hover:bg-[#001f7a] disabled:opacity-50 disabled:cursor-not-allowed flex items-center gap-1"
          >
            <Send size={14} /> 发送
          </button>
        </div>
      </main>
    </div>
  );
}
```

UI 要点：

- 单智能体时省略选择按钮区，直接展示该智能体描述
- 多智能体时顶部出现 chip 切换
- Enter 发送 / Shift+Enter 换行
- 流式中禁用发送按钮
- 加载失败/网络错误降级到错误气泡

## 6. 风险与缓解

| 风险 | 缓解 |
|------|------|
| middleware 分流误伤正常 PERSONAL 用户 | 判定条件 `payload.userType === "trial"` 严格匹配，不影响其他取值 |
| `/trial` 页面加载时立即 fetch，但 cookie 过期 | fetch `/api/trial/agents` 401 时 router.push 到 `/login` |
| SSE 在 Next.js dev server 流式延迟 | 已加 `Cache-Control: no-cache, no-transform`；dev 表现以 prod 标准为准 |
| 多智能体场景未来扩展时 chip 排版溢出 | 用 `flex-wrap` 容错 |
| 输入框 Shift+Enter 换行在某些 IME 下异常 | 留待节后回归，不在本期阻塞 |
