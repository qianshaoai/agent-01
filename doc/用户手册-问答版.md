# 用户使用手册（问答版）

> 本手册只讲普通用户怎么用平台，不涉及管理员功能。
> 用对话/问答形式写，按你最可能遇到的场景顺序排列。
> 看到不懂的字段可以 Ctrl+F 搜索。

---

第一部分 · 登录与首次使用

Q：我怎么登录？

A：浏览器打开平台地址。登录页有两个输入框：手机号、密码。

- 企业账号：手机号是单位管理员给你录入的；初始密码 = 你单位的"组织码"（比如 CESHI、DEMO2024）。第一次登录会强制让你改密码。
- 个人账号：自己点"注册账号"注册，手机号 + 自定义密码。

Q：我登不进去，提示"账号不存在，请先注册"，怎么办？

A：你的手机号还没被你单位管理员录入系统。两种方案：
1. 找你的单位管理员，让他把你加进去
2. 用页面右上角"注册账号"链接自己注册一个个人账号（个人账号有独立配额，但通常配额比单位的少）

Q：登录后我看到什么？

A：进入"人机协同工作舱"首页。从上到下三层：
- 顶栏（深蓝色）：左边是平台 Logo 和名字；右边一长串按钮 — 你的身份徽章、所属公司名、剩余配额（"剩余 XXX 次 · 至 YYYY-MM-DD"）、字体大小切换、帮助文档、联系我们、个人设置、退出
- 左侧栏：你的"我的工作流"列表，按工作流名称排列；最上面是"全部"
- 右侧主区：根据左侧选中的项展示内容（默认"全部"显示工作流卡片网格 + 你正在进行中的会话）

Q：什么是"剩余 XXX 次"？

A：你单位购买的额度，所有员工共用。每次跟智能体说一句话扣 1 次。如果是个人账号，则是你自己的额度。配额耗尽或过期前需要联系单位管理员/客服续费。

第二部分 · 工作流是什么

Q：什么是工作流？跟单纯找智能体聊天有啥区别？

A：工作流是一串按顺序排好的步骤，每一步可能让你跟不同的智能体聊、做人工审核、或调外部工具。设计目的是：把一项复杂的工作（比如"准备拜访政企大客户"）拆成清晰的几步，每一步专门给一个智能体处理一个子问题，最后所有步骤的产出汇集成完整成果。

如果你只是随便聊几句问问题，不需要走工作流。但如果你要完成一项有产出物的工作（写课程大纲、准备会议、整理客户资料），用工作流体验更连贯。

Q：工作流卡片上的"工作流"标签和"查看流程"是啥？

A：
- "工作流"标签：分类，告诉你这条流程归属哪个业务领域
- "查看流程"：点进去能看到这个工作流一共几步、每步是什么、用哪个智能体

Q：我点了"查看流程"进去了，怎么开始？

A：进入工作流详情页后，右上角有个蓝色按钮"开始新会话"。点它会弹一个命名框，让你给本次工作起个名（比如"项目 A 资料整理"），点确认就进入第一步。

第三部分 · 开始 / 继续 / 多个会话

Q：什么是"会话"？跟"工作流"是一回事吗？

A：
- 工作流（模板）：管理员搭好的，所有人共用的流程骨架，不变。
- 会话（实例）：你具体执行了一次工作流，叫一次"会话"。一个会话有它自己的名字、进度、对话记录。

举例：你这周用"拜访准备工作流"准备了 A 客户和 B 客户，那就是同一个工作流下的两个独立会话。两次会话的对话完全隔离，不会互相影响。

Q：怎么同时进行多个会话？

A：在工作流详情页，每点一次"开始新会话"就创建一个新会话。它们各自独立。回到首页能在"我的进行中工作流"区块看到所有进行中的会话，每条会话有"继续"、"重命名"、"关闭"按钮。

Q：刚刚没做完，关掉浏览器了，怎么继续？

A：重新登录后回首页。"我的进行中工作流"区块（标题在折叠/展开状态可切换）会列出你所有未完成的会话。点"继续"按钮就能从上次停的那一步接着做。

Q：会话太多了首页有点乱，怎么整理？

A：
- 进行中工作流默认显示 2 个，超过会出现左右切换箭头，滚动鼠标滚轮也能横向切换
- 不想要的进行中会话点关闭按钮把它放到历史里（不会真删除）
- 标题旁边的折叠按钮可以把整块进行中区域收起来

Q：会话名字想改，怎么办？

A：在首页的会话卡片上点铅笔按钮，弹窗里输入新名字回车保存。重命名只改名字，不影响对话记录和进度。

第四部分 · 与智能体对话

Q：进入第一步看到聊天界面，跟微信聊一样吗？

A：基本一样，但比微信多几个能力：

输入框下方一排小图标：
- 回形针：上传文件（PDF / Word / Excel / 文本 / 图片，最大 20 MB），AI 会自动读文件内容
- 麦克风：按住说话，松开自动转文字（需要单位管理员配置语音服务，未配置时会提示）
- 右边的纸飞机：发送

输入框里也支持：
- Shift+Enter 换行
- Enter 直接发送
- 粘贴图片（Ctrl+V）/ 拖拽图片进窗口
- 粘贴 Excel 表格会自动转为表格格式发送

Q：AI 回复的内容里有代码块，怎么复制？

A：鼠标移到代码块上，右上角会出现一个小复制按钮。点一下整段代码就到剪贴板了，不用拖蓝。

Q：AI 正在打字过程中我想打断它怎么办？

A：发送按钮位置在 AI 流式输出时会变成停止按钮，点击立即终止。停止后那条消息会显示"已停止"标记，但内容保留，下次发消息 bot 会跳过这条不当成上下文。

Q：我刚发的那条用户消息打错了，能改吗？

A：能。鼠标移到自己发的那条消息上，会出现编辑按钮。点开就地改，点保存会以新内容重新发送给 AI（旧的回复会被替换）。

Q：AI 这次回答得不好，能让它重新生成吗？

A：能。AI 最新一条回复下方有"重新生成"按钮，点击会用同一条用户消息再问一次，可能得到不同回答。

Q：聊天历史会保存吗？

A：会。你的所有对话都保存在数据库里。退出再登录也能看到历史。
- 在工作流会话里：聊天记录跟随该会话，不会跨会话泄漏（两个不同会话即使是同一个智能体，对话也互相独立）
- 不在工作流里：左侧"历史会话"列表能看到你跟该智能体所有对话

第五部分 · 走完一步去下一步

Q：跟 AI 聊够了我怎么进入下一步？

A：聊天页右上角有个蓝色按钮"下一步"。点它会弹个确认框：

> 即将进入下一步
>
> 当前步骤的对话内容会自动同步给下一步的智能体作为上下文参考。
>
> [取消] [继续]

确认后会跳到下一步，那个智能体已经"读过"你跟上一步聊的内容，能直接接着工作。

Q：什么是"下一步会带上一步的内容"？

A：举例：
- 第 1 步你跟"内容萃取师"聊了客户痛点
- 第 2 步进入"课程架构师"
- 课程架构师在你发第一条消息前，已经知道第 1 步的全部对话内容（系统自动注入），所以你不用重复说一遍

Q：如果当前步骤后面还有步骤但我点错"完成"了怎么办？

A：完成是最后一步才出现的按钮。中间步骤永远是"下一步"。如果你看到"完成"绿色按钮，说明现在已经是最后一步。点完成会弹"已完成所有工作流"庆祝弹窗 + 自动归档到历史。

Q：想回到上一步看看之前聊的怎么办？

A：聊天页左上角"上一步"按钮（同样在顶栏进度条旁），点击直接跳回上一步的聊天窗口，完整恢复当时的聊天记录（只读式查看自己之前的内容，可以继续聊）。

Q：进度条上每个圆圈是啥？颜色不同代表啥？

A：进度条横向排列所有步骤，每个步骤一个圆圈：
- 蓝色填充：当前步骤
- 绿色（已完成对勾）：已完成的智能体步骤（可点击回看）
- 黄色（已完成对勾）：已完成的人工步骤
- 灰色：尚未到达的步骤（不可点）

Q：我点了进度条上一个已完成的智能体步骤会怎样？

A：会把那一步的对话完整内容作为上下文注入给当前正在聊的智能体（不跳页）。这是为了你想"嘿，把第 2 步的产出再 review 一下"这种场景。

第六部分 · 人工步骤怎么处理

Q：我点"下一步"，跳到一个不是聊天界面，是个弹窗，说"这一步需要人工处理"，咋办？

A：这是人工步骤（人工执行 / 人工审核 / 外部工具）。流程是：
1. 弹窗里有"步骤说明"，告诉你需要做什么（比如"导出 Excel 表格交给同事审核"）
2. 你真去做这件事（在站外完成）
3. 做完回来点弹窗里的"已完成，继续"按钮
4. 系统自动帮你跳到下一步

Q：人工步骤要做什么有时候我看不懂，怎么办？

A：找你的工作流设计者（通常是组织管理员）问。每个工作流的步骤说明由管理员在后台填写，问他这一步具体含义。

Q：人工步骤不做行不行？

A：可以跳过——但跳过的话进度条上这一步会显示黄色对勾表示"已确认"，但实际上没有产出物。如果下一步的智能体依赖这个产出会出问题。建议老老实实完成。

第七部分 · 看历史

Q：上周做的工作流我想找回来看看，怎么找？

A：首页"我的进行中工作流"标题栏右边有"历史"按钮，点击进入"历史工作流"页面，列出你所有已完成或已放弃的会话。

Q：历史页面有什么操作？

A：
- 顶部搜索框：按会话名或工作流名搜
- 顶部筛选：全部 / 已完成 / 已放弃 切换
- 每条记录右侧两个按钮：
  - 查看：进入详情页，看每一步聊了什么
  - 永久删除：彻底从数据库抹掉，不可恢复（会弹确认）

Q：进入详情页能看到什么？

A：每个步骤一行，显示步骤标题、类型徽章（智能体/人工/审核/外部）。
- 智能体步骤：点"回看对话"进入只读聊天页，看当时的完整对话
- 人工/审核/外部步骤：显示"无对话"（这些类型本来就没有 AI 对话）
- 没进入过的步骤：显示"未对话"（已完成会话）/"未进入此步"（已放弃会话）

Q：进入"回看对话"后我能继续聊吗？

A：不能。这是只读模式：
- 顶部黄色提示"只读模式 · 历史会话回看"
- 输入框区域整块隐藏
- 消息上的"编辑 / 重新生成"按钮也隐藏（仅保留复制）
- "下一步 / 上一步"按钮隐藏

只能看，看完点左上角箭头返回历史详情页。

Q：进行中会话不要了怎么办？

A：在首页的会话卡片上点关闭按钮，确认后会软删除——从"进行中"消失，但变成"已放弃"状态进入历史，仍然可以回看。如果要彻底删，去历史页用"永久删除"。

第八部分 · 设置和其他

Q：字太小看不清怎么办？

A：顶栏右边那个 T 图标，点击会出现"小 / 中 / 大"三档。选了之后全平台所有页面字号、图标尺寸都会一起放大。设置会记住，下次登录还是你选的档位。

Q：怎么改密码？

A：顶栏齿轮图标 → 个人设置 → 修改密码。

Q：我换工作了不在原单位了，怎么把账号迁过去？

A：你不能自己改组织。让你新单位的管理员登录后台，去用户管理找到你的账号，用"修改所属组织"功能把你迁过去。迁完你重新登录就归属新单位了。

Q：我不想用这平台了，怎么删账号？

A：齿轮 → 个人设置 → 注销账号。会有 1 次确认。注销后你的对话历史会保留（用于审计），但你无法再登录。

Q：底部有个"联系我们"，是干啥的？

A：弹出一个二维码，扫码加管理员微信。如果遇到 bug 或者有功能建议、配额问题，从这里反馈最快。

Q：我看不到某个工作流，但同事能看到，咋回事？

A：每个工作流的可见范围由管理员设定，可能是：
- 只对你所在组织可见
- 只对你所在部门可见
- 只对某几个特定小组可见

如果你不在范围内，自然看不到。找你的组织管理员问，让他把你加到对应的部门/小组里。

第九部分 · 常见 trouble shooting

| 现象 | 可能原因 | 解决 |
|---|---|---|
| 发消息提示"使用次数已耗尽" | 你单位的配额用完了 | 找管理员充值 |
| 发消息提示"组织配额已到期" | 你单位购买的服务到期了 | 找管理员续费 |
| AI 长时间没响应 / 转圈 | 网络卡或上游服务超时 | 等 30 秒，仍无响应点停止后重发 |
| 上传图片报"文件过大" | 单图 > 10 MB | 压缩后再上传 |
| 上传文档报"文件过大" | 单文件 > 20 MB | 拆分或压缩 |
| 语音输入提示"暂未配置" | 单位还没接入语音服务 | 找管理员配置（VOLCENGINE） |
| 进度条停留在错误的步骤 | URL 参数过期 | 回首页点对应会话的"继续"按钮，重新进入 |
| 历史会话里看到不该有的对话 | 数据库异常或权限漏洞 | 截图发给管理员/客服 |
| "下一步"按钮点了没反应 | 浏览器扩展拦截或网络问题 | 关扩展、刷新页面 |
