# 智谱清言智能体配置说明

## 前置准备

### 1. 创建智能体

登录 [智谱清言](https://chatglm.cn)，进入**创作者中心** → **我的创作** → **新建智能体**，配置好智能体后发布。

发布后，在浏览器地址栏中找到智能体的 `assistant_id`，例如：

```
https://chatglm.cn/main/gdetail/6789abcd1234ef56
                                 ↑ 这串就是 assistant_id
```

### 2. 获取 API Key 和 Secret

进入**创作者中心** → **开发者面板** → **API Key 管理**，点击「新建」创建一组密钥。

页面会显示两列：

| 列名   | 说明                         |
|--------|------------------------------|
| Key    | 即 `api_key`，如 `4e981627c1398d03` |
| Secret | 即 `api_secret`，创建后只展示一次，请立即复制保存 |

> ⚠️ Secret 仅在创建时完整显示，关闭弹窗后将无法再查看，请务必妥善保存。

---

## 后台配置步骤

### 第一步：新增智能体

进入管理后台 → **智能体管理** → 点击右上角「新增智能体」，填写：

| 字段       | 填写内容                         |
|------------|----------------------------------|
| 智能体编号 | 自定义，如 `qingyan-case-helper`（全局唯一） |
| 名称       | 显示名称，如「案例助手」         |
| 平台       | 选择 `qingyan`                   |
| 分类       | 按需选择                         |

### 第二步：配置 API

创建后点击对应智能体的 🔑 图标，填写：

| 字段         | 填写内容                   |
|--------------|----------------------------|
| API Endpoint | 留空（使用默认地址）       |
| API Key      | 填写 Key，如 `4e981627c1398d03` |
| 模型参数     | 见下方 JSON                |

**模型参数（JSON 格式）：**

```json
{
  "api_secret": "你的 Secret 值",
  "assistant_id": "你的智能体 assistant_id"
}
```

填写示例：

```json
{
  "api_secret": "abc123def456",
  "assistant_id": "6789abcd1234ef56"
}
```

### 第三步：分配权限（可选）

点击 ⚙️ 图标，将智能体分配给需要使用的企业租户。不分配则仅管理员可见。

---

## 注意事项

- **每个开发者账号每天最多调用 5000 次**，总量不限，超限当天无法继续调用。
- **最多创建 5 个 API Key**，每个 Key 只能调用**自己账号创建**的智能体。
- 单次对话上下文支持 **32k token**（约 5.8 万汉字）。
- 系统会自动维护多轮对话上下文，无需额外配置。
- 如遇调用失败，先检查 `api_secret` 和 `assistant_id` 是否填写正确，以及智能体是否已**发布**。
