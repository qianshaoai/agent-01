平台使用手册

本手册重点讲清楚登录之后每个人能做什么、怎么做。按身份分章节看：你是哪种身份就读哪一段，不必从头读到尾。

身份分四种：普通用户、组织管理员、系统管理员、超级管理员。每往下一级权限更大。本手册里凡是写「管理员」如果不特指哪一级，就是后三种统称。

第一章 · 普通用户

登录入口：浏览器打开平台地址，手机号 + 密码登录。
- 个人账号：直接注册的那种
- 企业账号：找单位管理员要账号；第一次登录会要求改密码（必须改完才能进去）

进去之后看到的就是「人机协同工作舱」首页。

1.1 首页长啥样

顶部蓝色横条左边：平台 LOGO + 名称
顶部横条右边（从左到右）：
- 一个圆形徽章，写着你的身份（系统管理员 / 个人用户 / 企业用户等等）
- 一个小卡片，写着你所属企业名 + 企业代码（比如「前哨测试公司 CESHI」）
- 闪电徽章「剩余 多少 次 · 至 某日期」——这是你单位的总额度，所有员工共用
- 一个 T 字图标（字体大小切换）
- 一个书本图标（使用手册）
- 一个二维码图标（联系我们 / 客服）
- 一个齿轮图标（个人设置）
- 一个退出箭头

中间区域：
- 左侧栏：分类导航（「我的工作流 / 全部 / 高净值客户面谈准备 / 政企客户拜访 / AI 对话知识库 / 专属人格知识库 / 优秀实践案例萃取与撰写 / 企业培训课程开发」等等）
- 右侧大区：当前分类下的工作流卡片，每个卡片有图标、名称、一段描述、底部「查看流程」按钮

底部：版权信息 + 联系我们入口

1.2 工作流是啥

工作流就是一连串「按顺序要做的步骤」。每一步可能是：
- 打开某个智能体跟它聊天（最常见）
- 跑一个外部工具（比如生成 PPT、调用某个数据接口）
- 人工审核（要你自己看一眼然后点确认）

举例：「政企客户拜访准备工作流」可能 5 步：
第 1 步：跟「客户背景调研助手」聊，输入客户公司名，AI 给你输出公司近三年大事
第 2 步：跟「拜访目的梳理助手」聊，列你这次拜访想达成的 3 件事
第 3 步：跟「话术生成助手」聊，AI 根据前两步的内容生成开场白
第 4 步：人工审核，你看一遍话术，没问题点「确认」进下一步
第 5 步：跟「Q&A 预演助手」聊，模拟客户可能问的尖锐问题，AI 帮你想答案

走完一遍工作流，你就有一份完整的拜访准备资料。

1.3 怎么开始一个工作流

首页随便点一个工作流卡片的「查看流程」按钮 → 进入工作流详情页。

页面顶部是一行步骤卡片，每个步骤一个小卡片，按顺序排列。
- 已完成的步骤：绿色对勾
- 当前要做的步骤：蓝色高亮
- 还没到的步骤：灰色

每个步骤卡片底部都有一个按钮（默认显示「开始」「打开」「确认」之类，由管理员配置）。

点按钮 → 跳转到对应智能体的对话页面，或者执行外部工具，或者要求人工确认。

如果你只是想直接跟某个智能体聊（不走工作流），首页左侧切到「全部」或「我的智能体」分类，点智能体卡片直接进对话即可。

1.4 跟智能体聊天

进入对话页之后页面布局：

左侧栏「历史会话」：
- 你跟这个智能体聊过的所有对话，按时间倒序排
- 每条记录显示对话标题（默认取你说的第一句话前 20 字）+ 创建时间
- 点任意一条进入那段对话，能继续聊也能复制其中内容
- 顶部「+ 新建对话」按钮新开一段
- 中间有搜索框可以按关键字找历史

中间对话区域：
- 顶部：智能体名字 + 一个返回箭头 + 一个「对话导出」按钮（点了把整段对话存成 markdown 文件下载）
- 中间：消息流。你的消息靠右蓝色气泡，AI 的消息靠左白色气泡
- 底部：输入框

底部输入框周围的几个图标（从左到右）：
- 回形针：上传文件 / 图片
- 麦克风：按住说话录语音
- 输入框中间：打字
- 右下角飞机图标：发送

输入框上方有时会显示「引导问题」（管理员配的）—— 比如「你可以问我：xxx」「或者：xxx」。这些是建议你怎么问，点一下自动填进输入框。

发完一条消息：
- AI 开始流式吐字，你能看到它一个字一个字往外蹦
- 想中途停掉：点输入框右下角的停止按钮（发送按钮变成方块）
- AI 答完之后那条消息底部会出现几个小按钮：复制 / 重新生成 / 编辑（如果是你刚发的那条）

特别说明：
代码块复制
当 AI 给你输出一段代码（用三个反引号包起来的那种），代码块右上角悬停能看到一个复制按钮。点它只复制代码本身，不带前后那些解释文字。

聊天滚动
新消息进来不会强制把页面拉到底。你想边聊边往上翻历史完全可以，AI 还在吐字也不打断。如果想跟最新的，右下角有个向下箭头按钮一键回到底。

附件
回形针支持：
- 图片（jpg、png、webp 等）：会被压缩后传给 AI 看（前提是这个智能体是多模态模型，能看懂图）
- 文档（PDF、Word、TXT、Markdown 等）：会被自动提取出文字内容，跟你的消息一起发给 AI
- 单个文件 20MB 上限，超了要拆

语音输入
按住麦克风图标说话，松手自动转文字进输入框。背后是火山豆包语音模型，普通话识别率很高，方言可能不准。识别错了直接改输入框里的字就行。

1.5 个人设置

点顶部右上角的齿轮图标进入。

里面能改：
- 昵称：默认是手机号，可以改成真名或别的
- 密码：旧密码 + 新密码 + 确认新密码
- 头像：上传一张图
- 字体大小：小 / 中 / 大三档（也可以从顶部的 T 字图标直接改）
- 注销账号：彻底关闭这个账号，所有对话历史一并删除（慎用）

1.6 公告

第一次登录或者管理员发了新公告时，会弹一个公告对话框。看完点「我知道了」关掉。

公告分两种：
- 全平台公告：所有人都看（系统管理员或超级管理员发的）
- 本组织公告：只有你单位的人能看（你单位的组织管理员发的）

公告内容里可能有图片、链接、加粗等格式。链接点了直接跳转。

第二章 · 组织管理员

登录入口：在浏览器地址栏后面加 /admin，用管理员账号登录。

进去之后是「管理后台」的控制台，跟普通用户的工作舱完全是两个界面。左侧是侧边栏菜单，右侧是当前菜单的内容区。

侧边栏顶部显示：
- 平台 LOGO + 平台名
- 你的用户名 + 角色徽章（这里写「组织管理员」）

菜单分四组：

第一组「概览」
- 控制台
- 用量看板

第二组「组织与用户」
- 组织管理
- 用户管理

第三组「内容」
- 智能体管理
- 工作流管理
- 公告管理

第四组「系统」
- 操作日志

每一项点开就是右侧内容区的一个页面。下面分别说每一项怎么用。

2.1 控制台

进后台的第一眼。展示本组织（不是全平台）的概览数据：

顶部 4 个统计卡：
- 活跃组织（对你来说就是 1，因为只能看自己组织）
- 智能体数量（平台上能给你用的智能体总数）
- 总调用次数（你单位累计调用次数）
- 调用成功率（成功的请求占比）

下面 6 个导航卡：
- 组织管理 → 跳到组织管理页
- 用户管理 → 跳到用户管理页
- 智能体管理 → 跳到智能体管理页
- 公告管理 → 跳到公告管理页
- 用量看板 → 跳到用量看板
- 操作日志 → 跳到日志页

最下面：组织配额概览，本组织剩余额度的进度条。

2.2 组织管理

进来直接看到本组织一栏（不是列表，因为你只能管自己一个组织），并且自动展开。看到的信息：

主行：组织码、组织名、配额「已用 / 总数」、到期日、启用状态徽章。
（普通用户不可见的：组织管理员看不到「编辑 / 启停 / 删除」这些按钮——这些只有更高的管理员能动）

展开区域分两块：
组织结构
- 部门列表，每个部门一行，显示部门名、底下小组数
- 点部门左边的箭头展开看小组列表
- 每个部门行右侧有「编辑 / 删除」按钮
- 部门行底下有个输入框「输入部门名 + 添加」可以加新部门
- 进每个部门，里面有相同结构的「输入小组名 + 添加」

如果你是组织管理员，这里只能加 / 改 / 删部门和小组。改不了组织名 / 额度 / 到期日 / 启停 / 删除整个组织——这些只有超级管理员能动。

2.3 用户管理

本组织所有员工的列表。每行显示手机号、昵称、所属部门、所属小组、角色徽章、状态。

页面顶部有：
- 搜索框：按手机号、昵称、用户名找人
- 状态筛选：active（正常）/ disabled（禁用）/ deleted（已删除）/ trial（体验）
- 新增用户按钮

新增用户：点按钮弹出表单，填手机号 + 初始密码 + 选所属部门 + 选所属小组（可选）+ 角色（默认「普通用户」）→ 保存。该用户首次登录时会被强制要求改密码。

编辑用户：点用户行右侧的「编辑」图标，能改：
- 昵称
- 密码（管理员重置密码，会触发该用户下次必须重登）
- 状态（启用 / 禁用）
- 部门 / 小组（也叫「迁移到其他部门」）
- 角色：组织管理员只能往下调（普通用户互调、把人降级），不能调出比自己还高的角色

删除用户：点删除按钮，软删除（数据库标记为 deleted，对方下次登录提示「账号不存在」）。

修改所属组织（高级）：极少用。组织管理员通常没这权限，是系统管理员或超级管理员才有的功能——把员工从 A 组织挪到 B 组织。改了以后该员工会被强制踢出登录页重登。

用户分组：跟「部门 / 小组」是两套体系。
- 部门 / 小组：层级结构，一个员工只能在一个部门 / 一个小组
- 用户分组：扁平标签，一个员工可以同时属于多个分组（比如「市场部」+「VIP 客服组」+「英语流利」）
分组的用途：在工作流可见范围设置时，可以按「分组」精确分配权限。

2.4 智能体管理

组织管理员视角：能看到平台上所有的智能体，但每个智能体右侧标着「仅可查看」，新增 / 编辑 / 删除按钮全部隐藏。「分类管理」标签页也隐藏。

每个智能体卡片显示的字段：
- 名称 + 分类
- 平台（Coze、Dify、智谱、元器、清言、其他）
- 启用 / 禁用状态
- 智能体类型（站内对话型 / 外链跳转型）

外链跳转型智能体不在平台内对话，点首页卡片直接跳转到第三方网站。

2.5 工作流管理

组织管理员能看到本组织能用的所有工作流（不止你自己建的）。

页面顶部「工作流 / 分类」两个 tab。
- 工作流 tab：列出所有工作流卡片，每个卡片可展开看里面的步骤
- 分类 tab：管理工作流分类（增删改）

新建工作流：点右上角「+ 新增工作流」按钮，弹表单：
- 工作流名称
- 描述
- 分类（多选）
- 排序值（数字越小越靠前）
- 启用 / 禁用
- 可见范围：组织管理员只有「本组织可见」一种选择，蓝色提示框「本工作流将自动对您所在组织的所有成员可见（不可修改）」

保存工作流后，工作流是个空壳，需要再加步骤：

点工作流卡片展开看到一行「+ 新增步骤」按钮，点它弹步骤表单：
- 步骤标题
- 步骤描述
- 执行类型：
   智能体（agent）：点这步会跳到某个智能体对话
   外部工具（external）：跳到外部 URL
   人工审核（review）：要用户自己点「确认」才进下一步
   人工操作（manual）：跟 review 类似但措辞不同
- 绑定智能体（仅 agent 类型）：从下拉里选一个
- 按钮文字：默认「开始」可以改成「立即生成」「点我开聊」之类
- 启用 / 禁用

步骤可以拖动调换顺序、可编辑、可删除。

视图切换：每个工作流卡片右上角有「列表 / 流程图」切换。流程图视图把步骤画成一个流程图，便于直观看。

2.6 公告管理

发本组织专属公告。组织管理员发的公告只有本组织的人能看到。

页面是公告列表 + 新增按钮。新增公告表单：
- 标题
- 内容（支持 markdown：标题、列表、链接、图片、加粗等）
- 类型：普通 / 重要 / 紧急（影响显示样式：紧急是红色边框）
- 生效时间 / 失效时间（这段时间内会弹给目标用户）
- 目标范围：组织管理员只能选「本组织全员」，不能选「全平台」

公告发出后，所有目标用户下次进首页都会弹这条公告，关掉后下次不再弹（除非公告内容改了）。

2.7 操作日志

本组织所有员工的操作记录。每条记录字段：
- 时间
- 用户（手机号 / 昵称）
- 动作：login（登录）/ chat（聊天）/ upload（上传文件）/ speech（语音）/ workflow_step（工作流步骤）等
- 智能体（如果是 chat）
- 状态：success / error / aborted
- 耗时（毫秒）
- 错误信息（如果失败）

页面有筛选器：时间范围、动作、状态、用户。可以导出 CSV。

主要用途：
- 排查具体某个员工某次失败是什么原因
- 看高峰时段调用情况
- 追溯用户行为轨迹

2.8 用量看板

本组织各智能体的调用次数排行 + 时间趋势图。

主要看：
- Top 智能体：哪些智能体被本组织员工用得多（决定要不要给其它员工开同样的权限）
- 时间趋势：周 / 月 / 季的调用曲线
- 失败率：哪个智能体经常失败（找系统管理员排查）

第三章 · 系统管理员

跟组织管理员相比，系统管理员的视角是「全平台」而不是「单组织」：

3.1 哪些菜单和组织管理员看到的不一样

组织管理：能看到所有组织的列表，不只本组织。但一样不能创建 / 修改 / 删除组织本身——这些是超级管理员的活。

用户管理：能搜整个平台所有用户。能改任何用户的角色（往下调），能把用户从 A 组织迁到 B 组织。

智能体管理：从「仅可查看」升级为可改可删。
- 新增智能体：填名字、平台、API 端点、API Key、模型参数（JSON）、分类等
- API Key：会用 AES-GCM 算法加密落库，平台不会以任何形式回显或转发明文
- 编辑：改任何字段（除了 API Key 本身——只能整体覆盖更新）
- 删除：软删除，已经分配出去的关系一并清理
- 「分配给组织」按钮：把智能体分给哪些组织能用

工作流管理：能看跨组织的工作流，包括其他组织建的。

公告管理：能发全平台公告（所有用户都看到）。

操作日志：能查全平台所有调用日志，按组织 / 智能体 / 动作 / 状态多维筛选。

用量看板：跨组织聚合数据。每家公司用了多少额度、Top 智能体排行（全平台维度）。

3.2 系统管理员干不了的事

「品牌设置」菜单：超级管理员专属，系统管理员侧边栏里压根看不到这个入口。
创建 / 修改 / 删除组织：组织管理页里这些按钮对系统管理员是隐藏的。
改超级管理员的角色 / 密码：你管不了比你还高的人。

第四章 · 超级管理员

最高权限。除了系统管理员能干的所有事，再加：

4.1 品牌设置

侧边栏「系统」组里多一个「品牌设置」菜单。点进去能改：

平台 LOGO：上传一张图（建议 PNG 透明底，正方形），上传后所有用户的顶部 LOGO 立刻同步。
平台名称：默认「人机协同工作舱」，可以改成你公司想要的名字。
联系二维码：上传一张二维码图（一般是企业微信 / 客服微信），改完用户在「联系我们」入口能扫这个码。
版权信息：底部那行「© 2026 前哨科技」可以改。

这些改完是「立即生效」，所有正在使用的用户刷新就能看到。

4.2 创建 / 修改 / 删除组织

进「组织管理」，能看到所有组织的列表，每行右侧有「编辑 / 启停 / 删除」按钮（普通管理员看不到这些按钮）。

新增组织：点「+ 新增组织」，填：
- 组织码（4-8 位英文字母，会自动转大写，比如「DEMO」「CESHI」「ABCD」）
- 组织名（中文，比如「前哨测试公司」）
- 初始密码（员工首次登录企业账号时用这个密码）
- 配额（总调用次数，比如 500）
- 有效期（到这一天为止，过期之后这个组织所有人都用不了）

编辑组织：改组织名、改额度、改有效期、改启停状态、重置初始密码。

删除组织：要求组织内所有员工先迁走或删完才能删。删了之后这个组织的所有部门 / 小组 / 公告 / 权限规则全部级联清理，不可恢复。

4.3 跨身份操作

把任何人调成任何身份：包括把另一个超级管理员降为系统管理员、或者把普通用户提为组织管理员。

强制踢任何用户重登：用在你怀疑某个账号被盗用的时候。改了那个用户的密码或角色就会自动触发踢出登录页。

第五章 · 各种常见困惑

困惑一：管理员明明改了某人角色，他刷新页面没切过去？
答：刷新一下就生效，不用让他退出重登。如果还没切，再等几秒刷一次。

困惑二：员工被踢出去说「请重新登录」？
答：管理员对他做了下面任意一件事：改密码 / 改所属组织 / 改角色 / 禁用账号。这是设计如此，不是 bug。

困惑三：工作流创建出来别人看不到？
答：可见范围默认「仅自己」。要让别人看就改成「本组织可见」或「自定义」。组织管理员创建的工作流强制本组织可见，不能改。

困惑四：聊天聊到一半显示「网络错误」？
答：流式响应中断了。点「重新生成」再试。如果反复出错，先看顶部「剩余次数」是不是为零、有效期是不是过了。

困惑五：附件上传失败？
答：单文件限制 20MB。Word、PDF、TXT 等会被读出文字。图片会被压缩。超大文件先拆开。

困惑六：怎么导出整段对话？
答：对话页右上角「对话导出」按钮，点了存成 markdown 文件，离线也能看。

困惑七：语音输入识别错了？
答：松手自动发，识别错就再录或手动改输入框里的字。

困惑八：找不到某个智能体？
答：左侧切「全部」/「我的智能体」分类。还找不到，问组织管理员要不要把这智能体加到本组织。

困惑九：登录提示「该账号无后台访问权限」？
答：你只是普通用户，去平台首页登录，不要走 /admin。

困惑十：登录提示「该账号已被禁用」/「所属组织已过期」？
答：账号被关了或公司服务到期了。找管理员恢复或续费。

困惑十一：智能体说话答非所问 / 失忆？
答：智能体的「人格」是管理员配的提示词决定的，答得不准是提示词没调好。可以反馈给组织管理员让他调整智能体的提示词。失忆是因为对话太长被压缩了——这是为了控制成本（不然每次调用都把全部历史发过去会越来越贵）。

困惑十二：管理员配置智能体后用户用不了？
答：检查三件事——智能体是不是「启用」状态、是不是已经分配给该用户的组织、API Key 是不是配对了（有效期、余额）。

第六章 · 找谁求助

普通用户碰到问题：找单位的组织管理员
组织管理员碰到问题：找平台运维（系统管理员）
所有人都搞不定：找超级管理员

具体联系方式在平台首页底部 / 个人设置里有联系二维码（前提是超级管理员配过这个二维码）。

第七章 · 三件事记住就够

一、不同身份能干的事完全不同。看不到某个菜单不是 bug，是你权限就这样。
二、改了别人的角色 / 组织后，对方刷新就行，不用让他重登。
三、聊天前先看一眼顶部剩余次数和有效期，免得聊到一半被卡住。
